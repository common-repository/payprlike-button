== Description ==

Plugin create PayPRlike Button on your site.

== Installation ==

1. Upload the PayPRLike plugin to your blog and Activate it.
2. Go to Settings > PayPRLike Button and write your PayPRLike username in 'PayPRlike Username' field.
3. Go to Appearance > Widgets. Select PayPRLike widget and put in your preffered widget area.

That's all!
